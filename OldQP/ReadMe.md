# Solutions Or Comments For "Old" QP reports

## Description 

This is a repository of solutions and/or comments related to the "old" Quality Portal, i.e. the one located [here](https://quality.embarcadero.com/).

Each file is markdown, and named the same as the report number, e.g. [`RSP-10150.md`](RSP-10150.md) for [this report](https://quality.embarcadero.com/browse/RSP-10150).

## Submissions

If you'd like to contribute, please either file an [issue](https://github.com/DelphiWorlds/HowTo/issues), create a Pull Request, or [join the Delphi Worlds Slack workspace](https://slack.delphiworlds.com) and make your suggestion(s) there.
